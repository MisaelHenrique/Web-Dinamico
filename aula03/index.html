Arrays / Objetos String

  • caharAt
    • - retorna uma String contendo o caractere de posição
  • concat(string, string, ..., n)
    • - este metodo retorna uma string contendo um String n
  • replace
    • - busca na Stringuma suString que seja igual ao conteudo
  • slice(ini, fim)
    • - retorna a substring contento os carac, inicio até o fim

    • retorna um obj de array, contendo a subString entre o separator e o limit
  • subrstring(inicio, fim)
    • - retorna o resultado da conversão do objeto que o chamou o método (inicio até o fim)

    • - retorna o valor da string original convertido para minúsculo
  • toUpperCase()
    • - retorna o valor da string original convertido em letras maiúsculas

    • - remove o primeiro elemento do array
  • length
    • - propriedade que contem o tamanho do vetor
